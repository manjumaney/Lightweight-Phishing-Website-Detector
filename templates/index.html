How to use

  1. Enter the full link you want to review.
  2. Select Auto, Web Model, or URL Model.
  3. Click Check URL to start the analysis.
  4. In Auto mode, the system tries the Web model first and falls back to the URL model if needed.

How it works

The tool combines multiple signals to provide a more useful result than a simple safe or unsafe label.

🌐

Page-based review

When available, the system checks webpage-related signals and page content indicators for suspicious behavior.

🔗

URL-based review

If webpage retrieval is unavailable, the tool falls back to URL structure and lexical pattern analysis.

📊

Clear result summary

Each result includes a label, probability, risk level, and a short explanation of the signals involved.

About

This tool is designed to help users review suspicious links before opening them. It combines page-based checks, URL-based analysis, and a simple explanation layer to make results easier to understand.

It is intended to support safer browsing decisions and security awareness.

What the tool includes

  • Page-based phishing detection
  • URL-based fallback analysis
  • Risk level and score display
  • Probability chart
  • Recent check history

Disclaimer

This tool provides an automated assessment for educational and research purposes. Results are probabilistic and may not always reflect the final security status of a website. Use the output as a supporting signal, not as a guarantee.
